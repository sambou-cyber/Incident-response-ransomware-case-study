# Incident Response Documentation, Ransomware via Phishing (Healthcare)

## Overview
This project demonstrates my ability to analyze and document a cybersecurity incident using standard incident response practices. The scenario focuses on a simulated ransomware attack against a small U.S. healthcare clinic and highlights incident documentation, root cause analysis, and business impact.

## Scenario Summary
A healthcare clinic experienced a ransomware attack that disrupted business operations and prevented employees from accessing critical patient data. The attack originated from targeted phishing emails containing a malicious attachment, which deployed ransomware once executed.

- Identify attack vectors and root cause
- Assess operational and business impact
